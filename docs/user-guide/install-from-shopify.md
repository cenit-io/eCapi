# OMNA App installation process

**Find OMNA at Shopify store**

Go to your Shopify store administration, link Apps, and click on Shopify Store. 

<div align=center>
    <img width="800" src="/assets/images/installation/installation_0.png"/>
</div>

There, you could find out OMNA in the search box. Also you can go directly to [OMNA app](https://apps.shopify.com/omna-2?surface_detail=OMNA&surface_inter_position=1&surface_intra_position=3&surface_type=search)

<div align=center>
    <img width="800" src="/assets/images/installation/installation_1.png"/>
</div>

On the Add App button yoou will to start the installation process, and OMNA will open you a new tab with the installation process:

**Login**

OMNA will show you a login page where you should write your user and password for use OMNA app.

<div align=center>
    <img width="800" src="/assets/images/installation/installation_01.png"/>
</div>

After your enroll with user and password. OMNA will be installed in your Shopify Administration:

<div align=center>
    <img width="800" src="/assets/images/installation/installation_02.png"/>
</div>

Please, click on "Install App"

**First OMNA configuration**

After click Install App, OMNA will show you this firts page:

<div align=center>
    <img width="800" src="/assets/images/installation/installation_2.png"/>
</div>

This page indicates you that OMNA is configuring the initials settings to could run the app in your Shopify store.
It implies:
- configuration of the new user for use OMNA app.
- in your space, it is installing OMNA and Shopify collections with all its functionalities.
- principal Shopify flows configurations: Import products from an integration, Import stock-locations an integration, Import orders from an integration. [Go to Workflow list for more info](workflows)

This process could spent as normal one or two minutes of your time. After that, OMNA will open its homepage:

<div align=center>
    <img width="800" src="/assets/images/installation/home.png"/>
</div>

At this point OMNA is installed successfully in your Shopify store. If you go back to your Shopify administration, you could see OMNA installed.

<div align=center>
    <img width="800" src="/assets/images/installation/omna_installed.png"/>
</div>

If you open this new link, you will see an embebed application.

<div align=center>
    <img width="800" src="/assets/images/installation/dashboard.png"/>
</div>

This view will have a link to your OMNA workspace (Go to OMNA).

>To know how start to use OMNA app, please consult a [Quick Start](quick-start)
